## Вывод
> В рамках работы были использованы следующие виджеты:
> - Scaffold
> - AppBar
> - Center
> - Column
> - Text
> - SizedBox
> - Row
> - ElevatedButton
> 
> Обновление состояния реализовано функциями:
> - increment()
> - hugeIncrement()
> - reset()
> 
> Посредством параметров `onPressed` и `onLongPress`
